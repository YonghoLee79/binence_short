## 🎯 사용법 요약

### 🚀 빠른 시작 (3단계)
```bash
# 1. 가상환경 활성화
source venv/bin/activate

# 2. 환경 변수 설정 (.env 파일)

BINANCE_SECRET_KEY=your_secret_key
TELEGRAM_BOT_TOKEN=your_bot_token

# 3. 봇 실행
python hybrid_trading_bot.py
```
